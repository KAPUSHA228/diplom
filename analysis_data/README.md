
# Данные для мониторинга модели

## Структура файлов

### training/
- `students_autumn_2023.csv` - эталонные данные (на них обучалась модель)
- `students_spring_2024.csv` - данные следующего семестра

### monitoring/
- `monitoring_autumn_2023.csv` - признаки эталонного семестра
- `monitoring_spring_2024.csv` - признаки для проверки дрейфа
- `drift_demo.csv` - данные с искусственным дрейфом для демонстрации

## Как использовать

1. **Для первого анализа**: используйте autumn_2023 как эталон
2. **Для проверки дрейфа**: загрузите spring_2024 в мониторинг
3. **Для демонстрации**: загрузите drift_demo.csv - должен показать дрейф
